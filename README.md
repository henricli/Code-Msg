# 💬 Code-Msg

> Um lugar simples para conversar — mesmo quando tudo lá fora não estiver funcionando.

O **Code-Msg** é um site de mensagens inspirado no Discord, porém mais leve e direto.  
Ele foi criado principalmente para eu usar com meus amigos, mas está aberto para qualquer pessoa que queira um espaço simples para conversar.

A proposta é oferecer uma alternativa funcional e independente, permitindo comunicação mesmo em situações onde a internet tradicional não esteja tão estável.

---

## ⚡ Objetivo

- Criar um ambiente simples e fácil para conversar  
- Manter a comunicação entre amigos  
- Servir como alternativa ao Discord  
- Funcionar de forma leve e acessível para todos  

---

## 🧠 Ideia do Projeto

O Code-Msg nasceu com uma ideia simples:  
**e se um dia as grandes plataformas não funcionarem mais?**

Pensando nisso, o projeto foi desenvolvido para garantir que ainda seja possível conversar com amigos de forma prática e sem complicação.

Mesmo sendo um projeto pessoal, ele foi feito para qualquer pessoa que queira usar.
